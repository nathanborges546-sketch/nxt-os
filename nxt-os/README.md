# 🚀 NXT Command Center (Beta)

O **NXT OS** é uma plataforma proprietária de automação e gestão de prospecção B2B, focada em democratizar o acesso a fluxos de trabalho inteligentes (n8n, Make, Zapier).

## 🎯 Missão
Ajudar agências de marketing a escalar suas operações e aumentar o LTV através de automações de extração, qualificação e notificação de leads.

## 🛠️ Funcionalidades
* **NXT Purifier:** Módulo de higienização e consolidação de listas de leads CSV.
* **AI Diagnostics (RID):** Geração automática de diagnósticos técnicos usando Gemini 2.0 Flash.
* **One-Click Prospecting:** Interface de disparo para WhatsApp integrada ao CRM do Notion.

## 💻 Tech Stack
* **Interface:** Streamlit
* **Database:** Notion API
* **Intelligence:** Google Gemini
* **Processamento:** Pandas & Python

## ⚙️ Configuração Local
1. Instale as dependências: `pip install -r requirements.txt`
2. Configure o arquivo `.env` com suas chaves.
3. Execute: `streamlit run nxt_os.py`
